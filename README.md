# Superstore-sales-dashboard
## 📌 Project Overview
Analyzed 5,900+ retail sales records to uncover regional performance,
top products, category profitability and monthly growth trends.

## 🛠️ Tools & Technologies
| Tool | Purpose |
|------|---------|
| Python (Pandas) | Data cleaning & feature engineering |
| SQLite (SQL) | Data analysis & querying |
| Matplotlib | Exploratory visualizations |
| Power BI | Interactive dashboard |

## 📊 Key Insights
- **West region** generated the highest total sales
- **Technology** category had the highest profit margin
- **Q4** consistently showed peak sales every year
- **COD** was the most used payment mode
